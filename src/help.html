New Playback Speed option

Playback speed cannot work on pages where videos are actually hosted on a different site. (e.g. youtube videos not on the youtube site). This is a limitation of the Chrome browser itself.

When a site looks like it supports speed control, the toolbar icon will display a overlay

Now when clicked, the option so change playback speed will appear.

When any site loads their videos from a different site (e.g. a youtube video embedded in a non youtube site), then the speed control feature will not work and is disabled. Video Maximizer will fall back to the classic toggle behavior.

When I site doesn't support zooming, then a overlay will be displayed.

Hard-to-discover shortcuts

  1. The Unzoom function can be accessed by clicking this icon (or the Spacebar after opening)
  2. Clicking the speed-change button a second time toggles it back to normal speed.
  • TIP: ⭐⭐ Escape key on the page UNZOOMS. ⭐⭐
  • TIP: Pressing escape in the Speed Control window closes it.
  • NOTE: Speed changing will not work on sites hosting videos from other sites. This is just a limitiation of the Chrome APIs.
  • NOTE: there is an option to disable the Speed Controls completely (see Options below)

Pinning the extension to the toolbar makes it easier to use.

  1. To pin, first click on the Extensions puzzle piece icon
  2. Click the Pin icon

Changing options

  1. Right-click the icon
  2. Select Option from the dropdown menu. You can disable the speed controls and revert the behavior back to the classic toggle
