Universal Video Maximizer - version 3.0

Welcome to version 3.0.

This is a major rewrite that is required by the Google Chrome store upgrade to their latest API. Their goal is to improve security but there are still stability issues with this API.

You will only see this help message the first time you use the extension after it updates or installs.

  • This extension is FREE.
  • It requires minimal permissions to increase security.
  • There is NO server. NO DATA is ever uploaded anywhere. Your privacy is important.
  • It is Open Source project

Any problems should be reported here via a GitHub issue.

New Playback speed feature

Playback speed is a handy feature for speeding at 16x or slowing down playback on sites that may not support it.

Playback speed cannot work on sites where videos are actually hosted on different sites. (e.g. the site has videos that are hosted on youtube). This is a limitation of Chrome itself. "Cross site iframes" are intentionally blocked.

Hard-to-discover shortcuts

  1. The Unzoom function can be actived by clicking. Also Spacebar after opening
  2. Clicking the speed-change button a second time toggles it back to normal speed.
  • Keyboard: Pressing escape in the Speed Control window closes it.
  • Keyboard Pressing escape in the Zoomed Video on the Page unzooms it.
  • Warning Speed changing does not work on sites hosting videos from other sites.

Video Maximizer - Universal Tips

Pinning the extension to the toolbar makes it easier to use.

  1. Click on the Extensions puzzle piece icon
  2. Click the Pin icon

Changing options

  1. Right-click icon
  2. Select Option from dropdown menu. You can disable the speed controls and revert the behavior back to the classic toggle
