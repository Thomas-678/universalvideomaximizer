Universal Video Maximizer - version 3.0

Welcome to version 3.0.

This is a major rewrite that is required by the Google Chrome store upgrade to their latest API. Their goal is to improve security but there are still stability issues with this API.

You will only see this help message the first time you use the extension after it updates or installs.

  • This extension is FREE and Open Source
  • It requires minimal permissions to increase security.
  • Privacy is important. NO DATA is ever collected. The extension is also open sourced for transparency.

Any problems should be reported here via a GitHub issue.

New Playback Speed option

Changing the Playback Speed from 1/4x - 16x on sites that support it.

Playback speed cannot work on pages where videos are actually hosted on a different site. (e.g. youtube videos not on the youtube site). This is a limitation of the Chrome browser itself.

Hard-to-discover shortcuts

  1. The Unzoom function can be accessed by clicking this icon (or the Spacebar after opening)
  2. Clicking the speed-change button a second time toggles it back to normal speed.
  • Keyboard: Pressing escape in the Speed Control window closes it.
  • Keyboard Pressing escape in the Zoomed Video on the Page unzooms it.
  • Warning Speed changing will not work on sites hosting videos from other sites.
    This is a limitiation of the Chrome APIs.
    Note: there is an option to disable (see Options below)

Video Maximizer - Universal Tips

Pinning the extension to the toolbar makes it easier to use.

  1. To pin, first click on the Extensions puzzle piece icon
  2. Click the Pin icon

Changing options

  1. Right-click the icon
  2. Select Option from the dropdown menu. You can disable the speed controls and revert the behavior back to the classic toggle
